# A*寻路算法可视化工具

这是一个基于Windows API开发的A*寻路算法可视化工具，使用C++编写，提供了直观的图形界面来展示A*寻路算法的执行过程。

## 功能特性

### 🎯 核心功能
- **完整的A*算法实现**：支持8方向移动（4个正交方向 + 4个对角线方向）
- **实时可视化**：动态显示开放列表、已访问节点和最终路径
- **交互式地图编辑**：支持绘制墙壁、设置起点终点、擦除等操作
- **速度调节**：10级速度控制，可调整算法演示速度

### 🛠️ 工具选项
- **墙壁工具**：绘制障碍物
- **起点工具**：设置路径起点（绿色）
- **终点工具**：设置路径终点（红色）
- **擦除工具**：清除地图元素

### 📁 文件操作
- **保存地图**：将当前地图保存为二进制文件
- **加载地图**：从文件加载之前保存的地图
- **自动命名**：保存时自动生成包含时间戳的文件名

### 🎮 控制方式
**鼠标操作：**
- 左键点击：放置当前选择的工具
- 左键拖动：连续绘制/擦除
- 直线绘制：拖动时自动填充两点间的所有格子

**键盘快捷键：**
- `S` - 开始寻路
- `P` - 暂停/继续
- `T` - 停止寻路

**按钮控制：**
- 开始寻路、停止、暂停/继续
- 清空地图、随机地图
- 保存/加载地图

## 界面说明

### 地图区域
- 40×30 的网格地图
- 每个单元格大小为20×20像素
- 实时显示算法执行状态

### 图例说明
- 🟢 **绿色** - 起点
- 🔴 **红色** - 终点
- ⬛ **黑色** - 墙壁
- 🟡 **黄色** - 最终路径
- 🔵 **浅蓝色** - 已访问节点
- 🟢 **浅绿色** - 开放列表节点

### 信息面板
- 操作说明和使用提示
- 当前状态显示（运行中/已暂停/就绪）
- 起点终点设置状态
- 速度等级指示

## 技术实现

### 算法特点
- **启发式函数**：使用曼哈顿距离
- **移动代价**：正交移动代价10，对角线移动代价14
- **路径优化**：支持对角线移动，包含直角墙阻挡检测
- **内存管理**：完善的节点内存清理机制

### 系统要求
- Windows操作系统
- 支持标准Windows API
- 无需额外依赖库

## 使用指南

1. **设置地图**：使用工具绘制墙壁，设置起点和终点
2. **开始寻路**：点击"开始寻路"或按`S`键启动算法
3. **观察过程**：实时观察算法探索过程和最终路径
4. **调整参数**：使用滑块调整演示速度
5. **保存分享**：将有趣的地图保存下来或加载已有地图

## 构建说明

使用支持Windows API的C++编译器编译：
```bash
g++ -o AStarVisualizer.exe main.cpp -lcomctl32 -lgdi32
```

或者使用Visual Studio等IDE直接编译。
